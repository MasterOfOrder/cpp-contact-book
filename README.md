C++ Contact Book
A command-line contact manager built in C++. Contacts are saved to a file and persist between sessions.
Features

Add a contact (name, phone, email)
Display all contacts
Search for a contact by name (case insensitive)
Delete a contact
Contacts automatically save to a file and reload on startup

How To Compile & Run
Make sure you have a C++ compiler installed (g++ recommended).
bashg++ -o contacts main.cpp
./contacts
On Windows:
bashg++ -o contacts main.cpp
contacts.exe
Usage
1.Add Contact
2.Display Contacts
3.Search Contact
4.Delete Contact
5.Exit
Choose an option:
Enter the number of the action you want and follow the prompts.
